#### Mapping EF1 Components to EF2 

| Component                                        | EF1               | EF2                   |
| ------------------------------------------------ | ----------------- | --------------------- |
| Metadata, version, properties, screens, topology | `plugin.json`     | `extensions.yaml`       |
| Extension configuration                          | `plugin.json`     | `activationSchema.json` |
| Dependencies                                     | `plugin.json`     | `setup.py`              |
