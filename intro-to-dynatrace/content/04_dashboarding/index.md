## Dashboarding

Next we will create our first Dynatrace dashboard.

1. Open classic dashboards app
2. Create a new dashboard 
3. Add Headers for each section
    * Overview
    * Service
    * Infrastructure 
4. Overview
    * Add a problems tile
    * Add a health tile for hosts, services, databases, and applications
5. Service health
    * Add a data explorer single value tile
    * Create a chart for service response time
6. Infrastructure Health
    * Add a data explorer tile graph tile
    * Create a graph for host cpu usage
